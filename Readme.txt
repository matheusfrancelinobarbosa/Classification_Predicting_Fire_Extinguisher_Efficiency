ML Project (Classification):
Predicting Fire Extinguisher Efficiency.

Variavel Target (STATUS): 0 indicates the non-extinction state, 1 indicates the extinction state

In this project the R programming language and its libraries were used.
dplyr and plyr for data manipulation.
GGPLOT for graphical visualization.
For the ML models CARET and Random Forest package models were used.
ML models used: Logistic Regression and Random Forest.
